Tooltip Component

Documentation

Property Type Default Description
content string | TemplateRef 'Tooltip content !!!' The content to display in the tooltip. Can be a string or a template reference.
position 'top' | 'right' | 'bottom' | 'left' 'top' The position of the tooltip relative to the target element.
showArrow boolean true Whether to show an arrow pointing to the target element.
offset number 8 The distance between the tooltip and the target element in pixels.
zIndex number 1000 The z-index of the tooltip.
disableClose boolean false If true, the tooltip will not close when the mouse leaves the target.
timeout number 100 The delay in milliseconds before the tooltip closes when the mouse leaves.
alwayShow boolean true If true, tooltip always shows on hover. If false, tooltip only shows when text has ellipsis.
classContainer string '' Additional CSS classes to apply to the tooltip container.
classContent string '' Additional CSS classes to apply to the tooltip text content.
classTrigger string '' Additional CSS classes to apply to the trigger element.
disabled boolean false If true, the tooltip will not be displayed.

Function Controls

Method Description
showTooltip() Programmatically shows the tooltip.
hideTooltip() Programmatically hides the tooltip.
updatePosition(position) Updates the position of the tooltip. Position can be 'top', 'right', 'bottom', or 'left'.
