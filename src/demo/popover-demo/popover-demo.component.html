Popover Component

Documentation

Property Type Default Description
content TemplateRef undefined The content to display in the popover. Can be provided via content projection or template reference.
position 'top' | 'right' | 'bottom' | 'left' 'bottom' The position of the popover relative to the target element.
hasBackdrop boolean true Whether to show a backdrop behind the popover.
disableClose boolean false If true, the popover will not close when clicking outside.
disabledOpen boolean false If true, the popover cannot be opened.
blockInteraction boolean true If true, prevents interaction with elements behind the popover.
showArrow boolean true Whether to show an arrow pointing to the target element.
arrowAlignment 'center' | 'start' | 'end' 'center' The alignment of the arrow relative to the target element.
trigger 'click' | 'hover' 'click' The event that triggers the popover to open.
hoverRemoveDelay number 0 The delay in milliseconds before the popover closes when the mouse leaves (for hover trigger).
zIndex number 1000 The z-index of the popover.
offset number 8 The distance between the popover and the target element in pixels.

Function Controls

Method Description
openPopover() Programmatically opens the popover.
closePopover() Programmatically closes the popover.
togglePopover() Toggles the popover between open and closed states.
